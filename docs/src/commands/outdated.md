# stacy outdated

Check for package updates

## Synopsis

```
stacy outdated 
```

## Description

Compares installed package versions against the latest available from their
sources. Shows which packages have updates available without modifying anything.

## Examples

### Check for updates

```bash
stacy outdated
```
